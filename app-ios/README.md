## Get Started
### Setup
```sh
cd ../
./gradlew appioscombined:assembleXCFramework
```

### Open project with Xcode
```sh
 open DroidKaigi2022/DroidKaigi2022.xcodeproj
```

## Requirements
- Xcode 13.4.1 or Higher (13.4.1 in CI)
